# Venezuela Photography Expedition

An 18-day photography field plan for Venezuela (23 June – 10 July 2026), built for two photographers.

**Route:** Caracas → Orinoco Delta → Mérida (Andes) → Canaima & Angel Falls → Los Roques

Day-by-day, hour-by-hour, with verified internal flights, weather notes, golden-hour shot windows, health & safety, and packing.

Open **`index.html`** for the full interactive itinerary (also deployed live — see the repo's website link).

> A personal travel plan shared with friends. Reconfirm flights, fees, lodge bookings and the eVisa close to departure — Venezuelan domestic schedules change without notice. This is not professional travel advice.
